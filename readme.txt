This WordPress plugin protects the posts content from being copied by any other author (content copy protection). Blocks right mouse button, block Ctrl+C, Blocks possibility to select text on the page.

== Description ==

WP Text Copy Blocker is the security plugin to protect the posts content to being copied by another visitor of your website.

**Main features:**

* Block right click and context menu
* Disable the keys CTRL+A, CTRL+C, CTRL+X, CTRL+S or CTRL+V and etc
* Disable text selection on PC and mobile devices
* Disable Developers Tools (debugger)
* Collect statistics (by IP, Country, Visited page)
* Graphs by blocked countries (last 24 hours, last 7 days, last 30 days)
* Automatically block IP addresses
* Destroy content in browser of visitor
* It's easy to setup and free to use.
* Free support.


== Installation ==

1. Upload all the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
